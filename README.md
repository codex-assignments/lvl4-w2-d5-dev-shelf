# lvl4-w2-d5-dev-shelf

Objective Build a small full stack app with one clear user flow.

This app stores quick reference information and links to handy fullstack development tools. Add information to the left side, add as many links to resources as you like for it, then save the tool to the "shelf" which stores it in the backend api. The right side displays all current entries and will allow some editing and allows delete.

To run, backend must be running:
cd backend
pip install flask flask-cors
python app.py

Frontend:
new terminal, CLI
navigate to frontend folder
npm install
npm run dev
